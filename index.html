Overview

If you have an impressive music library and need a way to sort your files efficiently, you should probably consider turning to third-party software solutions, as they can help you achieve quick, convenient results. Rush is one of the applications that can simplify your work by enabling you to organize your audio files after various criteria.

Simplistic interface

This program comes with a minimalistic, user-friendly interface that packs a handful of neatly organized functions, which can be easily operated directly from the main window, without browsing endless menus looking for them. Although it does not feature a standard help manual, you can display a set of helpful tips by clicking the question mark icon. No additional configuration menu, window or pane is available, keeping this application as simple and accessible as possible.

Organize music files on your PC

Rush can help you organize your music library with minimum efforts by letting you move, rename and sort your data by various criteria. Before you begin the sorting process, you need to specify the source locations and a destination path on your PC. After doing so, you can toggle sorting criteria by toggling file types and defining a file order in the designated field. Among the supported formats, you can find MP3, M4A, AAC, FLAC, OGG and WMA. Setting the file order can be done by typing your desired order directly in the dedicated field, as guided by the brief instructions. It is possible that you can enable file overwriting, as well.

Generates logs

It is possible to view a log after the operation has been completed, which includes additional information such as a message and a message type. You can save the report as a file on your computer by using the Save to a File function.

Lightweight audio file organizer that can generate logs

To wrap it up, Rush is a handy file organizer that lets you sort your music library in a quick manner and generates a report afterward. It comes with a simple user interface that packs a handful of accessible functions and although it lacks standard help documentation, you are provided with a brief set of instructions, which can help you understand its purpose better.
