Rush - Music File Organizer

Rush is an simple tool that will allow you to organize your music files by moving, renaming, and placing them into the directory structure of your choice based on the file order.

Features

# the ability to choose the order of your folder structure.
# to select how the files are named.
# name files as you want.
# no harm to files.
# detailed log.
# simple interface
